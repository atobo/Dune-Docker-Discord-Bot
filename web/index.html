Discord Bot Configuration

Configure your Discord Bot token here. The backend container will automatically read these values.

How to setup your Discord Bot

  1. Go to the Discord Developer Portal and click New Application.
  2. On the General Information page, copy your Application ID and paste it into the Client ID field below.
  3. Go to the Bot tab, click Reset Token, copy the new token, and paste it into the Discord Token field below.
  4. Scroll down on the Bot page to Privileged Gateway Intents and toggle ON all three intents (Presence, Server Members, Message Content). Click Save Changes.
  5. Go to OAuth2 -> URL Generator, select the bot and applications.commands scopes, and select Administrator permission. Open the generated URL to invite the bot to your server!
